# Dual-model Memory Compression

## 核心概念

双模型记忆压缩技术，通过双模型协作实现高效的上下文记忆管理。

## 关键特性

- **80% Token节省**：大幅降低上下文开销
- **5分钟滚动压缩**：定时压缩机制，保持记忆新鲜度
- **智能去重**：识别并合并重复信息

## 应用场景

- 长对话上下文管理
- 多轮任务记忆优化
- 资源受限环境部署
