## 解决方案

风格
   - Discord/WhatsApp: No markdown tables, use bullet lists
   - Discord links: 多个链接用<>包裹避免embed
   - WhatsApp: No headers, 用bold或CAPS强调

3. 反应（React）使用
   React when:
   - 欣赏但不需要回复（👍, ❤️, 🙌）
   - 觉得有趣或发人深省（🤔, 💡）
   - 想确认但不打断对话（✅, 👀）
   - 简单yes/no或批准（✅, 👀）

   Don't overdo it: 每个消息最多一个reaction
