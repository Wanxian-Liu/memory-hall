# 知识分类体系（TAXONOMY）

## 一、顶层分类（6类）

| 类型 | 说明 | 示例 |
|------|------|------|
| skill | 技能（可执行操作、方法、技巧） | 烹饪技能、编程技能 |
| document | 文档（说明性内容） | 教程、说明文档 |
| concept | 概念（理论、原理、定义） | 算法原理、设计模式 |
| rule | 规则（约束性内容） | 安全规则、业务流程 |
| pattern | 模式（可复用经验/案例） | 设计模式、解决方案 |
| workflow | 工作流（流程性内容） | CI/CD流程、审批流程 |

## 二、二级标签体系

### skill
- 技术技能 / 软技能
- 领域技能（天工/九重天/记忆殿堂）

### document
- 教程 / 指南 / API参考 / 架构文档

### concept
- 理论 / 原理 / 定义 / 术语

### rule
- 安全规则 / 业务规则 / 约束条件

### pattern
- 设计模式 / 代码模式 / 解决方案

### workflow
- 开发流程 / 审批流程 / 运维流程

## 三、分类维度（Taxonomy）

| 维度 | 二级标签 |
|------|---------|
| 技术 | 编程/框架/工具/系统 |
| 学习 | 教程/实践/理论 |
| 天工 | 熔炉/析构/创造 |
| 项目 | 进度/问题/决策 |
| 产品 | 需求/设计/迭代 |
| 运营 | 数据/用户/内容 |
| 日常 | 日程/任务/笔记 |
| 记忆殿堂 | 萃取/归一/围栏 |

## 四、置信度阈值

| 阈值 | 行为 |
|------|------|
| ≥0.90 | 直接使用 |
| 0.70-0.89 | 使用但待确认 |
| 0.50-0.69 | 多候选 |
| <0.50 | 未分类 |
