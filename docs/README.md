# 记忆殿堂v2.0

> 完美记忆系统 - 融合claw-code设计的分布式持久记忆解决方案

## 📖 项目概述

记忆殿堂v2.0 是一个高性能、安全可靠的分布式持久记忆系统，专为AI代理设计。它提供了语义搜索、内容萃取、去重归一化、智能分类等核心功能，并集成了五级权限模型和审计日志系统。

### 核心特性

| 特性 | 描述 |
|------|------|
| **WAL协议** | Write-Ahead Log三段式提交，确保数据一致性 |
| **通感搜索** | 语义向量搜索，支持近似最近邻检索 |
| **萃取模块** | 智能内容提取与摘要生成 |
| **归一化** | SimHash去重，冲突检测与解决 |
| **分类系统** | 多维度知识分类 (skill/document/concept/rule/pattern/workflow) |
| **安全围栏** | 六层安全防御，空间隔离与权限管控 |
| **观自在监控** | 六维健康指标，自适应阈值监控 |
| **CLI工具** | 交互式命令行界面 |
| **插件系统** | 可扩展的插件架构 |

## 🏗️ 系统架构

```
记忆殿堂v2.0
├── gateway/          # 网关核心模块
├── base_wal/         # WAL协议实现
├── permission/      # 权限引擎
├── config/          # 配置系统
├── sensory/         # 通感语义搜索
├── extractor/       # 内容萃取
├── normalizer/      # 去重归一化
├── classifier/      # 智能分类
├── fence/           # 安全围栏
├── health/          # 观自在监控
├── cli/             # 命令行工具
├── plugin/          # 插件系统
├── task/            # 任务管理
├── audit/           # 审计日志
├── public/          # 公共记忆空间
└── private/         # 私有记忆空间
```

- **五级权限模型**: READONLY → READ → WRITE → DANGER_WRITE → DANGER_FULL_ACCESS
- **六层防御体系**: 身份认证、工具白名单、参数验证、敏感操作确认、频率限制、审计日志
- **围栏隔离**: 公共空间与私有空间分离
